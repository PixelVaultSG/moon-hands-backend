## 5-Minute Customization

### 1. Update Branding

Edit `/mnt/okcomputer/output/app/src/App.tsx`:

```javascript
// Change business name (line ~50)
const demoConversation = [
  { speaker: 'ai', text: 'Hello! This is Sarah from [YOUR BUSINESS NAME]. How can I help you today?' },
  // ...
];
```

### 2. Update Pricing Currency (if needed)

```javascript
// Change $ to SGD if targeting Singapore only
<span className="text-4xl font-bold">${plan.price}</span>
// to:
<span className="text-4xl font-bold">S${plan.price}</span>
```

### 3. Update Contact Info

Find the booking dialog and update:

```javascript
// Around line 500+
<DialogContent>
  <DialogTitle>Book a Demo Call</DialogTitle>
  // Add your contact info here
</DialogContent>
```

### 4. Rebuild & Redeploy

```bash
cd /mnt/okcomputer/output/app
npm run build
# Then redeploy the dist folder
```
