## TECHNICAL NOTES FOR VIDEO PRODUCTION

### Screen Recording Setup
- **Device:** iPhone 15 Pro (for sharp display) or simulator
- **WhatsApp:** Use WhatsApp Business API sandbox or mock interface
- **Telegram:** Use actual Telegram bot interface
- **Resolution:** 1080x1920 (9:16 vertical)
- **Frame rate:** 30fps

### Timing
- Total: ~68 seconds (with transitions)
- Each WhatsApp message appears with 0.3s stagger
- Typing indicators show for 1.5–2s before response
- Transitions: 0.5s slide between WhatsApp ↔ Telegram

### Text Appearance Style
- WhatsApp green bubbles: #25D366 (patient) / #DCF8C6 (clinic AI)
- Telegram: standard Telegram light theme
- Overlay text: White with subtle dark shadow for readability
- Font: SF Pro (iOS system font) for authenticity

### Hardcoded Text Rationale

| Element | Why This Text |
|---------|--------------|
| "HIFU" | Most popular aesthetic treatment in Singapore; clinics immediately recognize |
| "S$388" | Realistic mid-market price point |
| "Dr. Chen" | Common Singapore doctor name |
| "277 Orchard Road" | Prime aesthetic clinic location; creates aspiration |
| "Saw us on Instagram" | Most common patient acquisition channel |
| Emoji usage | Matches how real Singapore patients message |
